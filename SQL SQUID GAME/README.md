## 🎮 Welcome to SQL Squid Game

Think you can survive the games?

This repository contains my solutions to the **SQL Squid Game** challenges. Each level introduces a new scenario inspired by the Squid Game universe while testing different SQL concepts ranging from basic filtering to advanced analytical queries.

Every challenge is designed around a fictional dataset where you'll solve real-world SQL problems using progressively more advanced techniques.

# 🚦 Level 1 — Red Light Green Light

### 🎯 Mission

Identify vulnerable players who are:

- 🟢 Alive
- 💰 In severe debt
- 👴 Elderly **OR**
- 🎲 Gambling addicts with no close family

# 🍱 Level 2 — Rations

<img width="993" height="477" alt="image" src="https://github.com/user-attachments/assets/39904f43-10fa-4284-a7a8-5ad01880c2ca" />

### 🎯 Mission

Calculate the number of food rations required for **90% of surviving non-insider players** and determine whether the available food supply is sufficient.

- Aggregate Functions
- COUNT()
- FLOOR()
- CASE
- CROSS JOIN
- Boolean Logic

# 🍯 Level 3 — Honeycomb

### 🎯 Mission

Analyze the average completion time for each honeycomb shape during the **hottest and coldest months** over the **last 20 years**.

### 📖 SQL Concepts

- Common Table Expressions (CTEs)
- Date Functions
- EXTRACT()
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries


---

# 🪢 Level 4 — Tug of War

### 🎯 Mission

Calculate the average age of teams with exactly **10 players**, categorize them into age groups, and rank them based on average age.

### 📖 SQL Concepts

- HAVING
- CASE
- AVG()
- GROUP BY
- Window Functions
- RANK()
